# LeetCode
Solutions to Leetcode problems

## Typescript

Create a new solution + test file
```
npx ts-node .\GenerateSolutionTemplate.ts
```

Run tests for a single solution
```
yarn jest Easy/0001.tests.ts
```

Run all tests
```
yarn jest
```

## Dotnet
Run Tests for a single solution
```
dotnet test --filter Tests.Easy.Tests0013
```

Run all tests
```
dotnet test
```
